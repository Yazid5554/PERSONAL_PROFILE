Kenapa Saya Terjun ke Dunia Web Development?

Perjalanan saya dimulai saat SMK, ketika pertama kali melihat website dan bertanya-tanya "Bagaimana cara membuatnya?". Rasa penasaran itu membawa saya untuk belajar HTML pertama kali dari YouTube.

Yang membuat saya jatuh cinta dengan programming adalah kemampuannya untuk mengubah ide menjadi kenyataan. Setiap baris kode yang saya tulis adalah langkah untuk menciptakan sesuatu yang bermanfaat bagi orang lain.

Saya percaya bahwa teknologi web adalah masa depan, dan saya ingin menjadi bagian dari perubahan itu. Setiap project yang saya kerjakan adalah kesempatan untuk belajar dan berkembang lebih baik.
